2020.04.02

* command + space : 검색 
* command + shift + 4 : 화면 캡처(사용자가 범위 지정)
* command + shift + 5 : 화면 캡처 또는 화면 녹화 
* command + R : 새로고침
* (크롬) command + 클릭 : 새 탭으로 열기
* command + control + Q : 화면 잠금
* option + shift : 단어 단위로 선택
* command + shift : 한 줄 선택
* (Xcode) control + . : 코드 자동완성
